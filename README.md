## Installation:

Ok First Of All Let's Get The HaxeFlixel's Shit
--------------------------------------------------
1. First of all [Install Haxe](https://haxe.org/download/) or if it doesn't work [Install 4.3.4](https://haxe.org/download/version/4.3.4/)
2. Then after installing [Haxe](https://haxe.org/) [Install Flixel/HaxeFlixel](https://haxeflixel.com/documentation/install-haxeflixel/)
3. Install git

□ Windows: Access [This Link](https://git-scm.com/install/windows) and press install for x64 git

□ Linux: install the git package: sudo apt install git (ubuntu), sudo pacman -S git (arch) and blablabla you probably have this part

4. Setupping Some Shit

□ haxelib setup

AND DONE WOW

5. Install and set up your libraries!

□ Just go to the setup and run windows.bat

and if some dependencies failt to install via bat you can simply install the core shit with these:

□ haxelib install lime 8.1.2

□ haxelib install openfl 9.3.3

□ haxelib install flixel 5.6.1

□ haxelib install flixel-addons 3.2.2

□ haxelib install hscript

6. Compiling Your Game

Use Powershell/Terminal idk which cmd you got and get to the root of the source

then run this command

Windows:

□ lime test windows

Linux:

□ lime test linux

7. You Finished!
